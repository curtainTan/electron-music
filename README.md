使用本项目

第一步安装依赖

```bash
npm install
或
yarn
```
这里建议使用yarn安装，因为需要下载electron浏览器内核，有54m+，蛮大的，如果用npm安装会非常慢

本地运行项目

```bash
npm start
```

项目打包成 App

```bash
npm run dist
```
